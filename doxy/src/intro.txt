\section intro1 Document Overview
\par
The present document constitutes Version 1.0 of the Developer's
Guide for KLone, developed by <a href="http://www.koanlogic.com">KoanLogic S.r.l.</a>.
It describes how to develop an end-user application based on the KLone
framework.
\par
KLone runs under POSIX compliant systems, but is designed to be highly 
portable to other operating systems.

The KLone library is described in detail in the 
<a href="modules.html">KLone API</a> section, where
the functions are grouped in software modules according to their 
purpose. From an object-oriented point of view, each module represents
a <em>class</em>. The library contains all the functions or <em>methods</em>
that allow:

\par
For each software module mentioned above, the function documentation does not 
include the description of the internal data structures, which are irrelevant
for the end-user. The library includes all the functions that make data
manipulation transparent to the programmer, hence enforcing the information
hiding criteria as a design approach.
All the provided functions are documented
from the application programmer's point of view, specifying the function 
prototypes, the required input parameters, and the returned codes.
